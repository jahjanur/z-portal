# Custom fonts (e.g. Panchang)

Place your font files here, then ensure `index.css` @font-face paths match.

**Suggested filenames for Panchang:**
- **Variable (one file):** `Panchang-Variable.woff2`
- **Static weights:** `Panchang-Regular.woff2`, `Panchang-Medium.woff2`, `Panchang-Bold.woff2`

Supported formats: `.woff2` (preferred), `.woff`, `.ttf`, `.otf`.
